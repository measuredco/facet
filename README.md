# Facet

Measured abstract brand image generator.

## Run locally

Install dependencies:

```bash
pnpm install
```

Start the dev server and open the local URL shown in terminal:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```

## Notes

- p5 is installed via `pnpm` and bundled with Vite.
